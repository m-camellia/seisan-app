# 🎡 精算アプリ

テーマパークや旅行での割り勘・精算を計算するWebアプリです。

## バージョン

| バージョン | URL | 説明 |
|-----------|-----|------|
| v1 | [v1を開く](https://seisan-app.pages.dev/v1/) | 立替払いが1人の場合 |
| v2 | [v2を開く](https://seisan-app.pages.dev/v2/) | 立替払いが複数人の場合 |

## 機能

- 項目ごとに割り勘メンバーを選択
- 手動調整で端数を調整
- 最小取引数で精算方法を自動計算
- データはブラウザに自動保存
- iPhoneのホーム画面に追加して使用可能
